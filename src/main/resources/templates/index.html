Projeto A3 2024.1
Estruturas de Dados e Análise de Algoritmos
Busca e ordenação usando dados da Last.FM

Membros:

Código-Fonte do projeto no GitHub

O projeto tem como objetivo buscar todos os dados salvos no banco de dados e utilizar os algoritmos internos para filtrar os dados por usuário e ordenar por música, artista, álbum ou data, além de utilizar um algoritmo de busca binária para encontrar músicas pelo nome.
