Busca e ordenação usando a API do Last.FM

Membros:

O projeto tem como objetivo buscar as faixas ouvidas de um determinado usuário da Last.fm, filtrando por data e ordenar os dados de modo a determinar um ranking dos artistas, faixas e genêros mais ouvidos pelo usuário em determinado período. Os dados são salvos em um banco de dados SQLite e são usados para comparar ranks passados.
